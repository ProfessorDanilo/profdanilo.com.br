Arduino

Arduino

O Arduino consiste em um hardware (placa com circuito eletrônico) que permite que você controle um microcontrolador e execute muitas rotinas com ele. Como exemplo, você pode acionar atuadores como ligar e desligar um LED, girar um motor, abrir ou somente destravar uma porta etc. Também pode ler sensores como sensores de temperatura, pressão, inclinação, magnéticos e muitos outros.

Estou com um projeto para traduzir alguns comandos do Arduino para o Português e se você curtir esta ideia, fique a vontade para me ajudar.

Neste site, vou disponibilizar a biblioteca para você baixar, colocar um help sobre as funções que traduzi e adicionarei vídeos com eventuais exemplos de como utilizar tais comandos.

Biblioteca

Baixe a biblioteca

Para começar a usar a biblioteca, basta fazer o download do arquivo profDanilo.zip e, no IDE do Arduino, vá em Sketch -> Adicionar Arquivo e clique no arquivo .zip que você baixou.

Se você quiser me ajudar a traduzir a biblioteca, peça a solicitação para editar esta biblioteca no GitHub.
