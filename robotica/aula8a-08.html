Abaixo você encontra o código da aula de hoje.

/*
PWM
controla a luminosidade de um led conforme o valor do potenciômetro
*/

 int ledPin = 3; // pino do led
 int analogPin = 0; // pino para leitura do potenciômetro
 int val = 0; //variável para armazenar o valor lido

 void setup()
 { 
    pinMode(ledPin, OUTPUT); // configura pino como saída
 }

 void loop()
 {
    val = analogRead(analogPin);  // le o valor analógico
    analogWrite(ledPin, val / 4); // aciona led com o valor analógico lido
                                   //dividido por 4 para ajustar ao valor  
                                   //máximo que pode ser atribuído a função
}

Vimos em aulas passadas que o Arduíno possui uma porta analógica de 10 bits de entrada. Isso quer dizer que o Arduíno lê uma tensão que varia de 0 até 5V e transforma em um número inteiro que vai de 0 até 1023.

Note que 2 x 2 x 2 x 2 x 2 x 2 x 2 x 2 x 2 x 2 = 210 = 1024. Ou seja, existem 1024 valores, mas como contamos com o 0, então a faixa de valores possíveis é de 1023. Note então que fazemos 2 número de bits - 1 para sabermos o intervalo numérico quando a porta possui um certo "número de bits".
