POTENCIÔMETRO

Nesta aula, vamos ver o funcionamento de um potenciômetro que servirá para controlar o uma tensão elétrica em determinado dispositivo.

Podemos usar tal dispositivo para controle da saída de áudio de uma caixa de som, posição de um motor, brilho de uma lâmpada entre outras possibilidades. As conexões para tanto não são tão simples, então, vamos começar com um circuito que nos permite controlar o brilho de um LED.

A folha usada em aula servirá de base para montarmos o nosso circuito.

Nesta aula não teremos atividades para serem entregues.
