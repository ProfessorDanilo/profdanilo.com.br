Aula extra

No dia de hoje, o professor entrou na sua turma, entretanto não fomos para o laboratório, uma vez que não tinha nenhum projeto pronto/planejado (foi uma aula extra).

Assim, a aula foi utilizada para falar como montar um circuito e programar um Arduino usando uma simulação: a plataforma Tinkercad. Lá o professor aproveitou para mostrar como converter o código em blocos para código em texto e assim permitir que você programe utilizando a linguagem Scratch.

Para utilizar o simulador, acesse esta sala no Thinkercad e use o nick aluno.

Em breve, o professor irá começara a solicitar os códigos das aulas seguintes, na forma de texto, antes de iniciarmos a aula.
